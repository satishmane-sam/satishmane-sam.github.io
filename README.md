## 🗂️ Repository layout

```
├── index.md          ← Live homepage (what visitors see)
├── about.md          ← About page
├── _posts/           ← Blog articles (one file = one post)
├── _layouts/         ← Page templates
├── _includes/        ← Custom styles & head
└── _config.yml       ← Site configuration
```
